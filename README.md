# sdt-test
Assessment Techincal Test for PT Surya Digital Teknologi 

## Installation

1. Clone the project

```bash
  git clone https://github.com/widz11/sdt-test.git
```

2. Go to the project directory

```bash
  cd sdt-test
```

3. Setup env database & queue "email"

```bash
  cp enc-example .env
```

4. Install dependencies

```bash
  npm install
```

5. Run migration

```bash
  npm run migration:run
```

6. Start the server

```bash
  npm run start
```

7. Go to service

```bash
  http://localhost:3030
```

8. Run test

```bash
  npm run test
```
